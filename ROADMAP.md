# Roadmap

## Сделано

### Ядро
- [x] Авторизация и навигация по Moodle (campus.fa.ru)
- [x] Сканирование курсов, сбор тестов/заданий/преподавателей
- [x] Решение тестов (radio, checkbox, shortanswer, essay) через Groq
- [x] Написание эссе-заданий через Groq
- [x] Запись лекций через МТС Линк API
- [x] Удержание присутствия на вебинаре (PresenceKeeper)
- [x] Telegram-бот с подтверждением перед каждой отправкой
- [x] Поддержка dry-run (`/preview`)

### Качество и надёжность
- [x] Кэш вопросов/ответов (`qa_cache.py`) — не тратим API на повторы
- [x] Оценка после сдачи теста (`get_quiz_grade`)
- [x] Дайджест теста (`quiz_digest.py`) — карточка к сессии
- [x] Напоминания о дедлайнах (за 24ч и 1ч)

### Многоплатформенность
- [x] Поддержка online.fa.ru как второго Moodle-инстанса
- [x] `UniBrowser` параметризован `base_url`
- [x] `/scan_online` — отдельное сканирование online.fa.ru

### Знания
- [x] `KnowledgeMirror` — SQLite-трекинг уверенности по темам
- [x] `/learn` — микро-квиз по слабой теме с семантической оценкой ответа
- [x] `/map` — карта знаний: слабые/сильные темы

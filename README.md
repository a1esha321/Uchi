# 🎓 ИИ-агент для заочного обучения

Автоматический агент для учёбы на заочном:
- Посещает прямые эфиры МТС Линк и накапливает конспекты
- Сам находит все курсы на сайте университета
- Решает тесты на основе лекций  
- Читает требования преподавателей и пишет ответы на задания
- Управляется через Telegram с подтверждением перед отправкой

---

## Быстрый старт на Railway

### 1. Подготовь ключи
- **Groq API**: https://console.groq.com → Create API Key
- **Telegram бот**: @BotFather в Telegram → /newbot
- **Telegram Chat ID**: напиши @userinfobot

### 2. Деплой
1. Railway → New Project → Deploy from GitHub
2. Выбери свой репозиторий
3. Variables → добавь все переменные из `.env.example`
4. Дождись деплоя

### 3. Первый запуск
В Telegram боте:
```
/start    — справка
/scan     — бот сам найдёт все твои курсы на сайте
/subjects — список найденных предметов
```

## Как пользоваться

### Авто-сканирование сайта
```
/scan
```
Бот зайдёт на сайт под твоим логином, найдёт все курсы, тесты и задания. Предметы создадутся автоматически.

### Лекции (через расписание)
Перешли боту сообщение с расписанием (с синими ссылками на семинары):

```
18:55-20:25. Иванов И.И. — в.з./д.а..
Основы коммуникации (семинар).
```

Бот распарсит время и спросит какой предмет привязать. Запустит лекцию автоматически в нужное время.

### Тесты
Отправь прямую ссылку на тест с сайта. Бот:
1. Спросит какой предмет
2. Решит тест используя конспекты
3. Пришлёт ответы на подтверждение с кнопками
4. Только после твоего OK — отправит на сайт

### Задания
Отправь ссылку на страницу задания. Бот:
1. Прочитает требования преподавателя
2. Напишет ответ используя конспекты
3. Пришлёт черновик на подтверждение
4. После OK — вставит и сохранит на сайте

### Пакетная обработка
```
/quizzes     — все тесты по всем предметам
/assignments — все задания по всем предметам
```

## Структура проекта

| Файл | Назначение |
|---|---|
| `telegram_bot.py` | Главный бот (точка входа) |
| `agent.py` | Логика агента: лекции, тесты, задания |
| `browser.py` | Управление браузером (Playwright) |
| `lecture_listener.py` | Слушание эфиров через API МТС Линк |
| `presence.py` | Открытие страницы эфира |
| `smart_solver.py` | Решение тестов (Groq) |
| `essay_solver.py` | Написание ответов (Groq) |
| `subjects.py` | Реестр предметов + база знаний |
| `telegram_notifier.py` | Уведомления и подтверждения |

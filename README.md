# 🎓 ИИ-агент для заочного обучения

Автоматический агент для campus.fa.ru (Moodle):
- Посещает прямые эфиры МТС Линк и накапливает конспекты
- Сам находит все курсы, тесты, задания, требования преподавателей
- Решает тесты с подтверждением через Telegram
- Пишет текстовые ответы на задания
- Управляется из Telegram

## Команды бота

**Основные:**
- `/scan` — просканировать все курсы (использует переменную SEMESTER)
- `/subjects` — список предметов с флагами
- `/course <id>` — карточка конкретного предмета
- `/upcoming` — ближайшие дедлайны из календаря
- `/quizzes` — пройти все тесты с подтверждением
- `/assignments` — выполнить все задания

**Дополнительные:**
- `/preview <url>` — dry-run: решить, но НЕ отправлять
- `/stats` — статистика работы бота
- `/export <id>` — скачать все конспекты предмета
- `/teachers` — список преподавателей
- `/debug <url>` — отладка страницы
- `/debug_test <url>` — отладка теста

## Флаги предметов

- ✅ **completed** — сдано (предмет прошлого семестра)
- 🔗 **external** — на другой платформе (online.fa.ru)
- 📝 **needs_enrollment** — нужна запись на курс

Бот автоматически пропускает предметы с этими флагами.

## Деплой на Railway

1. Fork репозитория на GitHub
2. Railway → New Project → Deploy from GitHub
3. Variables → добавь всё из `.env.example`
4. Railway использует Dockerfile автоматически

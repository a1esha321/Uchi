# 🎓 ИИ-агент для заочного обучения

Агент автоматически посещает прямые эфиры на МТС Линк,
накапливает конспект лекции через субтитры и сдаёт тесты
на основе реальных знаний из лекции.

---

## Как это работает

```
Эфир МТС Линк
      ↓
Агент открывает страницу → платформа видит студента
      ↓
Каждые 5 минут забирает субтитры → накапливает конспект
      ↓
После лекции — сохраняет конспект в файл
      ↓
Открывает тест → Claude отвечает на основе лекции
      ↓
Сдаёт тест
```

## Установка (делается один раз)

### 1. Установи Python 3.11+
Скачай с https://python.org — при установке отметь **"Add to PATH"**

### 2. Создай виртуальное окружение
```bash
cd uni_agent
python -m venv venv

# Windows:
venv\Scripts\activate

# Mac / Linux:
source venv/bin/activate
```

### 3. Установи зависимости
```bash
pip install -r requirements.txt
playwright install chromium
```

### 4. Создай файл `.env`
Скопируй `.env.example` → переименуй в `.env` → заполни своими данными:
```
ANTHROPIC_API_KEY=  ← получи на console.anthropic.com
MTS_TOKEN=          ← получи в МТС Линк → Настройки → API
UNI_URL=            ← адрес сайта университета
UNI_LOGIN=          ← твой логин
UNI_PASSWORD=       ← твой пароль
```

## Настройка (важно!)

### Шаг A — Найди ID эфира МТС Линк
Открой ссылку на эфир — в URL будет число:
```
https://events.mts-link.ru/12345678
                              ↑
                           это event_id
```

### Шаг B — Настрой селекторы своего сайта
Открой `browser.py` и найди комментарии `⚠️ НАСТРОЙ`.

Чтобы найти правильные селекторы:
1. Открой сайт университета в Chrome
2. Нажми **F12** → вкладка **Elements**
3. Нажми иконку курсора (или Ctrl+Shift+C)
4. Кликни на поле логина → скопируй селектор
5. Вставь в `browser.py`

### Шаг C — Заполни `main.py`
```python
attend_lecture_and_pass_quiz(
    webinar_url="https://events.mts-link.ru/ТУТ_ТВОЙ_ID",
    event_id="ТУТ_ТВОЙ_ID",
    quiz_url="https://университет.ru/quiz/123",  # или None
    duration_minutes=90  # длительность лекции
)
```

## Запуск

### Вручную (для теста):
```bash
python main.py
```

### По расписанию (постоянный режим):
Открой `scheduler.py`, настрой дни и время лекций, затем:
```bash
python scheduler.py
```

## Структура файлов

```
uni_agent/
├── .env              ← твои секреты (не отправляй никому!)
├── .env.example      ← шаблон для .env
├── requirements.txt  ← зависимости
├── main.py           ← главный файл, запуск здесь
├── browser.py        ← управление браузером
├── lecture_listener.py ← слушает лекцию через МТС Линк API
├── presence.py       ← держит страницу открытой
├── smart_solver.py   ← Claude решает тесты на основе лекции
├── scheduler.py      ← автозапуск по расписанию
└── notes_XXXXX.txt   ← конспекты лекций (создаются автоматически)
```

## Частые проблемы

**Субтитры не приходят**
→ Проверь тариф МТС Линк — субтитры могут быть недоступны на базовом тарифе

**Селекторы не работают**
→ Открой `browser.py`, найди `⚠️ НАСТРОЙ` и замени селекторы на свои (через F12)

**Ошибка API ключа**
→ Проверь что `.env` файл заполнен правильно и находится в папке проекта

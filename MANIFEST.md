# Uchi — Manifest

Автоматизированный агент для заочного обучения на базе Moodle (campus.fa.ru, online.fa.ru).

## Что умеет

- **Сканирование курсов** — обходит Moodle, собирает тесты, задания, требования преподавателей
- **Решение тестов** — парсит вопросы (radio, checkbox, shortanswer, essay), отвечает через Groq LLM с учётом конспектов лекций
- **Написание заданий** — генерирует эссе на основе задания и требований преподавателя
- **Запись лекций** — слушает эфиры МТС Линк, сохраняет конспекты
- **Дайджест теста** — после каждого сданного теста генерирует карточку знаний
- **Микро-квизы** — `/learn` задаёт вопрос по слабой теме, трекает уверенность
- **Карта знаний** — `/map` показывает сильные и слабые темы
- **Напоминания** — предупреждает о дедлайнах за 24ч и 1ч
- **Кэш ответов** — не тратит API-квоту на одинаковые вопросы

## Принципы

- Один LLM-провайдер: Groq (llama-3.3-70b-versatile)
- Подтверждение пользователя перед отправкой любого теста или задания
- Все данные локально: JSON-реестры, SQLite, текстовые конспекты
- Headless Chromium для автоматизации браузера (Playwright)
